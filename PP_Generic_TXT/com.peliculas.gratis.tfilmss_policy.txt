TFilmss Privacy Policy
TFilmss Privacy Policy:
In the privacy policy of TFilmss we inform you how and for what TFilmss uses the user data during the use of this App.
- The data of the user never is stored in a database neither own nor external and this data is not used for any action other than the correct operation of the application locally.
- All permissions requested by the app are to store information locally that helps the user to access their content quickly and conveniently and never to save their data as stated in the first point.
- The only internet connections made by the user through our App are for access to the Youtube API that provides the videos and for the connection with ad networks for the monetization of the App.
Politica de privacidad de TFilmss:
En la política de privacidad de TFilmss informamos de cómo y para qué usa TFilmss los datos de cada usuario duarante el uso de dicha App.
- En ningún momento ningun dato del usuario es guardado en una base de datos ni propia ni ajena ni es usado para ninguna otra acción que no sea el correcto funcionamiento de la aplicación de forma local.
- Todos los permisos pedidos por la app son para guardar información de forma local que ayude al usuario a tener acceso a su contenido de forma rápida y cómoda y nunca para guardar sus datos como se expone en el primer punto.
- Las unicas conexiones a internet que realiza el usuario mediante nuestra App son para el acceso a la API de Youtube que provee los videos y para la conexión con redes de anuncios para la monetización de la App.
