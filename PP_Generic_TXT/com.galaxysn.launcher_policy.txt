Following are the explaination for the used permissions:
Read the contents of your USB storage
This permission allows users to view our product wallpaper and store them on your phone.
Set an alarm
This permission allows users to access the device's alarm through our product clock widget.
Read Google service configuration
This information is used to acquire advertising ID, our product provides users with better advertising service by using such anonymous ID.
Full network access
This permission is used to access the device's network for certain functions including receiving update notifications, retrieving AD.
Retrieve running apps
This permission is used in our Booster feature in order to view the running apps.
Run as startup
Our product may run as startup to provide users with a launcher service.
Install shortcuts
This permission is used to install shortcuts on desktop when users install new apps, this is normal feature for a launcher product.
Modify system settings
This permission is used in our product settings, in order to switch or adjust ringtone, vibration and brightness level of the screen.
Draw over other apps
This permission allows the our Sidebar feature to be displayed as a floating window on other apps.
