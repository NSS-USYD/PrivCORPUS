Requested permissions of Android device
Read external storage:
used to allow the user to edit audios saved in the device.
Write external storage:
used to save edited audios by the user.
Read contacts:
used to allow the user to read the list of contacts when assigning a tone notification.
Write contacts:
used to allow the user to assign a tone notification to a contact.
Write settings:
used to allow the user to set a default tone alarm or tone notification to the device.
Record audio:
used to allow the user to record his voice and used it as tone notification or tone alarm.
Internet:
used by AdMob for advertising.
