About user data and Cloud Backups
When using HabitNow as a non-premium user we do not store any of the user data in any server. All data provided to HabitNow is only stored in your device and our development team can not access to it.
We use a third-party cloud based service to store premium user backups. Backup files are stored pseudonimously and its data is obfuscated. These files are protected by security rules that prevent the access of unauthorized or malicious users. We will never share or sell its data to any third-party entity.
Cloud backups are an optional feature. You can delete your backed up data from the app at anytime.
