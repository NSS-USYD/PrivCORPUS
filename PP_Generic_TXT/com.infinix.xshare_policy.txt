What Information Do We Collect?
1.XShare Collects location data to enable features as followed when App is in use
1.1 Open Camera to Scan QR Code for Transfer Connection
When the app is in use, XShare will collects location data to open camera and enable transfer connection, when users scan the QR code with the camera. It is necessary permission required by system camera, so that users devices can decode QR code and users can start transfer files through XShare. However, XShare would never record or upload location information under such circumstance.
1.2 Use Location Permission to Open Bluetooth to Discover Nearby Devices.
When the app is in use, and users receiving files, XShare would collects location data location permission to enable discover of nearby devices through BluetoothManager scanning. It is a crucial step to connect between devices. However, XShare never record or upload location information under such circumstance.
1.3 Use Location Permission to Create wifi Hotspot for Transfer.
When the app is in use, XShare would collects location data location permission to enable creation wifi group or hotspot group for transfer. XShare will never record and store any location information under such circumstance.
1.4 Extend Location data Usage to Ads
Facebook and admob are two third party advising SDK. Aming at improving better experience for users, Xshare would extend the location data usage to support facebook and admob advertising, so that users can receive personalized ads and decrease irrelevant ads when app is in use.

When App is in use, and also when app is in the background, Athena SDK, a self developed SDK for behaviour analysis, would collect user location data to enable users portraits analysis. It does help in differentiating users and providing more appropriated content.
