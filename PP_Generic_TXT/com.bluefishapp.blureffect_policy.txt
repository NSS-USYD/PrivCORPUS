Privacy Policy
Personal Information
Blur Effect app does not violate any kinds of users privacy. This app does not collect any personal information such as user name, email address, mobile number or precise geographic location.
Sensitive Permissions
In order to use this app it needs some sensitive users permissions such as...
1. Internet access (to show ad and fetch remote data )
2. Device storage read permission ( to read media file when you want to work with them)
3. Device storage write permission (to write media file when you want to save your working file )
